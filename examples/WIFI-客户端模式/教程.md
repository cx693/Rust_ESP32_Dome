## 📖 前言

本篇实现 ESP32-S3 连接路由器 WiFi，通过 DHCP 获取 IP 地址，同步 NTP 时间，并启动一个 HTTP Web 服务器实现远程 LED 控制。

### 你将学到

- WiFi STA（客户端）模式连接路由器
- DHCP 自动获取 IP 地址
- NTP 时间同步协议
- Embassy TCP 服务器实现
- Web API 设计（RESTful）

### 一、WiFi STA 连接

```rust
// 配置 WiFi 客户端模式
let station_config = Config::Station(
    StationConfig::default()
        .with_ssid("Cudy-AC89")           // 路由器名称
        .with_password(String::from("123513as")),  // 密码
);

// 创建 WiFi 驱动
let (controller, interfaces) = esp_radio::wifi::new(
    peripherals.WIFI,
    ControllerConfig::default().with_initial_config(station_config),
).unwrap();

// 使用 embassy-net 管理网络栈
let (stack, runner) = embassy_net::new(
    interfaces.station,
    embassy_net::Config::dhcpv4(Default::default()),  // DHCP 自动获取 IP
    mk_static!(StackResources<3>, StackResources::<3>::new()),
    seed,
);

// 等待获取 IP
stack.wait_config_up().await;
if let Some(cfg) = stack.config_v4() {
    rprintln!("Got IP: {}", cfg.address);
}
```

### 二、NTP 时间同步

```rust
async fn sync_ntp(stack: Stack<'static>) -> Option<u32> {
    // 创建 UDP socket
    let mut socket = UdpSocket::new(stack, ...);
    socket.bind(0).ok()?;

    // 构造 NTP 请求包（48 字节）
    let mut pkt = [0u8; 48];
    pkt[0] = 0x1b;  // NTP v3, client mode

    // 发送到 NTP 服务器
    let server = IpEndpoint::new(
        IpAddress::Ipv4(Ipv4Address::new(162, 159, 200, 123)),  // Cloudflare NTP
        123,
    );
    socket.send_to(&pkt, server).await.ok()?;

    // 接收响应
    let mut resp = [0u8; 128];
    let (n, _) = socket.recv_from(&mut resp).await.ok()?;

    // 提取时间戳（字节 40-43，大端序）
    let ts = u32::from_be_bytes([resp[40], resp[41], resp[42], resp[43]])
        .wrapping_sub(2_208_988_800);  // NTP epoch → Unix epoch
    Some(ts)
}
```

**NTP 协议原理：**
```
ESP32                    NTP Server (162.159.200.123)
  │                           │
  │──── NTP Request ─────────→│  (48 字节，几乎全 0)
  │                           │
  │←─── NTP Response ────────│  (48 字节，包含 Unix 时间戳)
  │                           │
  提取字节 40-43 → Unix 时间戳
```

### 三、HTTP 服务器

```rust
loop {
    // 创建 TCP socket
    let mut socket = TcpSocket::new(stack, &mut rx_buf, &mut tx_buf);
    socket.set_timeout(Some(Duration::from_secs(10)));

    // 等待客户端连接
    socket.accept(80).await?;

    // 读取 HTTP 请求
    let req = read_request(&mut socket).await;
    let (method, path) = parse_request(&req);

    // 路由处理
    match (method, path) {
        ("GET", "/") => {
            // 返回 HTML 页面
            respond(&mut socket, "200 OK", "text/html", HTML.as_bytes()).await;
        }
        ("GET", "/api/status") => {
            // 返回 LED 状态 JSON
            let json = status_json(LED_ON.load(Ordering::Relaxed), now_ts);
            respond(&mut socket, "200 OK", "application/json", json.as_bytes()).await;
        }
        ("POST", "/api/toggle") => {
            // 切换 LED
            let new = !LED_ON.load(Ordering::Relaxed);
            LED_ON.store(new, Ordering::Relaxed);
            led.set_level(if new { Level::Low } else { Level::High });
            respond(&mut socket, "200 OK", "application/json", ...).await;
        }
        _ => {
            respond(&mut socket, "404 Not Found", ...).await;
        }
    }

    socket.flush().await.ok();
    socket.close();
}
```
