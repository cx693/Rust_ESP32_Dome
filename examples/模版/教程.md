## 📖 前言

本项目是一个 **最小化模板**，包含 ESP32-S3 Rust 裸机开发的最少代码，可以作为创建新项目的起点。

## 🧩 项目结构

```

```

## 📝 模板代码

```rust
#![no_main]
#![no_std]

use esp_bootloader_esp_idf;
use esp_hal::{
    delay::Delay,
    gpio::{Input, InputConfig, Level, Output, OutputConfig, Pull},
    main,
};
use panic_rtt_target as _;
use rtt_target::{rprintln, rtt_init_print};

esp_bootloader_esp_idf::esp_app_desc!();

#[main]
fn main() -> ! {
    rtt_init_print!();
    rprintln!("模板项目启动");

    let peripherals = esp_hal::init(esp_hal::Config::default());

    // ── 在这里添加你的代码 ──
    // 示例: LED + 按键
    let mut led = Output::new(peripherals.GPIO48, Level::Low, OutputConfig::default());
    let key = Input::new(peripherals.GPIO0, InputConfig::default().with_pull(Pull::Up));

    let mut status: bool = false;
    let delay = Delay::new();

    loop {
        if key.is_low() {
            status = !status;
            rprintln!("状态: {}", status);
            led.set_level(if status { Level::Low } else { Level::High });
            delay.delay_millis(120);
            while key.is_low() {}
        }
    }
}
```

## 🚀 使用方法

### 1. 复制模板

```bash
cp -r examples/模版 examples/我的项目
cd examples/我的项目
```

### 2. 修改 Cargo.toml

```toml
[[bin]]
name = "我的项目"    # ← 改成你的项目名
path = "./src/bin/main.rs"
```

### 3. 编写代码

编辑 `src/bin/main.rs`，添加你的功能。

### 4. 编译烧录

```bash
cargo run
```

## 📋 常用外设模板

### GPIO 输出（LED）

```rust
let mut led = Output::new(peripherals.GPIO48, Level::Low, OutputConfig::default());
led.set_high();   // 输出高电平
led.set_low();    // 输出低电平
led.toggle();     // 翻转电平
```

### GPIO 输入（按键）

```rust
let key = Input::new(peripherals.GPIO0, InputConfig::default().with_pull(Pull::Up));
if key.is_low() { /* 按下 */ }
if key.is_high() { /* 松开 */ }
```

### 延时

```rust
let delay = Delay::new();
delay.delay_millis(500);   // 毫秒
delay.delay_micros(100);   // 微秒
```

### RTT 打印

```rust
rprintln!("Hello {}", 42);
rprintln!("调试信息: {:?}", some_value);
```

## 🔍 知识点总结

| 知识点 | 说明 |
|--------|------|
| `#![no_std]` | 禁用标准库 |
| `#![no_main]` | 禁用标准 main |
| `esp_app_desc!()` | ESP-IDF 引导加载程序描述符 |
| `esp_hal::init()` | 初始化芯片硬件 |
| `loop {}` | 嵌入式主循环 |
| `rprintln!()` | RTT 调试输出 |
