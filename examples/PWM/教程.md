## 📝 核心代码讲解

### 一、PWM 原理

PWM 通过快速切换高低电平来模拟"中间亮度"：

```
占空比 0%:   ____________  → 全灭
占空比 50%:  _██__██__██_  → 半亮
占空比 100%: ████████████  → 全亮

██ = 高电平（LED 亮）
__ = 低电平（LED 灭）

频率 5kHz → 人眼看不到闪烁
```

### 二、LEDC 配置

```rust
// 1. 创建 LEDC 控制器
let mut ledc = Ledc::new(peripherals.LEDC);
ledc.set_global_slow_clock(LSGlobalClkSource::APBClk);  // 80MHz 时钟

// 2. 配置定时器（决定 PWM 频率和分辨率）
let mut timer0 = ledc.timer::<LowSpeed>(timer::Number::Timer0);
timer0.configure(timer::config::Config {
    duty: timer::config::Duty::Duty10Bit,  // 10 位分辨率 → 0~1023
    clock_source: timer::LSClockSource::APBClk,
    frequency: Rate::from_khz(5),           // 5kHz 频率
}).unwrap();

// 3. 配置 3 个通道，分别驱动 R/G/B
let mut ch_r = ledc.channel(channel::Number::Channel0, peripherals.GPIO4);
ch_r.configure(channel::config::Config {
    timer: &timer0,
    duty_pct: 0,                    // 初始占空比 0%
    drive_mode: DriveMode::PushPull,
}).unwrap();
```

**定时器参数说明：**
```
时钟源: APB = 80MHz
分辨率: 10bit → 2^10 = 1024 级
频率: 5kHz

PWM 周期 = 1 / 5000 = 200μs
每级时间 = 200μs / 1024 ≈ 0.195μs
```

### 三、呼吸灯曲线

用整数抛物线替代浮点 sin 函数：

```rust
fn breathe(ch_r, ch_g, ch_b, color, delay) {
    for step in 0..=BREATH_STEPS {
        let t = step as u32;
        let s = BREATH_STEPS as u32;
        // 抛物线: 4*t*(s-t)/s → 形似正弦半波
        // t=0 → 0, t=s/2 → s, t=s → 0
        let brightness = (4 * t * (s - t)) / s;
        let pct = ((brightness * 100) / s) as u8;

        // 按比例缩放各颜色分量
        let r = (r_base as u32 * pct as u32 / 100) as u8;
        let g = (g_base as u32 * pct as u32 / 100) as u8;
        let b = (b_base as u32 * pct as u32 / 100) as u8;

        ch_r.set_duty(r).unwrap();
        ch_g.set_duty(g).unwrap();
        ch_b.set_duty(b).unwrap();
        delay.delay_millis(20);
    }
}
```

**曲线对比：**
```
亮度
 ▲
 │      ╭──╮           sin: 需要浮点运算
 │    ╭╯    ╰╮         抛物线: 纯整数，嵌入式友好
 │  ╭╯        ╰╮       效果几乎一样！
 │╭╯            ╰╮
 └──────────────────→ 时间
 0    0.5    1.0
```

### 四、7 色循环

```rust
const COLORS: [(u8, u8, u8); 7] = [
    (100, 0, 0),      // 红
    (0, 100, 0),      // 绿
    (0, 0, 100),      // 蓝
    (100, 100, 0),    // 黄 (红+绿)
    (0, 100, 100),    // 青 (绿+蓝)
    (100, 0, 100),    // 品红 (红+蓝)
    (100, 100, 100),  // 白 (红+绿+蓝)
];

loop {
    for &(r, g, b) in &COLORS {
        // 纯色常亮 1 秒
        ch_r.set_duty(r).unwrap();
        ch_g.set_duty(g).unwrap();
        ch_b.set_duty(b).unwrap();
        delay.delay_millis(1000);

        // 呼吸灯效果
        breathe(&mut ch_r, &mut ch_g, &mut ch_b, (r, g, b), &delay);
    }
}
```

## 🔍 知识点总结

| 知识点 | 说明 |
|--------|------|
| PWM | 脉冲宽度调制，用占空比控制亮度 |
| LEDC | ESP32 的 LED PWM 控制器，硬件生成 PWM |
| 定时器 | 决定 PWM 频率和分辨率 |
| 通道 | 将 PWM 信号输出到指定 GPIO |
| 占空比 | 高电平时间 / 周期，0%=灭，100%=亮 |
| 抛物线呼吸 | 4t(s-t)/s 替代 sin，纯整数运算 |
