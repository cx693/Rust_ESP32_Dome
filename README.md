# Rust_ESP32_Dome
A collection of Rust bare-metal examples for ESP32 using the `esp-hal` framework. Covers GPIO, UART, I2C, SPI, ADC, PWM, WiFi (AP/STA/sniffer), BLE, and a 3D cube renderer on ST7789 display. Each example is a standalone Cargo project with Embassy async support. Ideal for learning embedded Rust development on ESP32.
